# HR-ANYLIST-DASHBOARD-EXCEL
# HR Analytics Dashboard

An interactive HR Analytics Dashboard built in Microsoft Excel to analyze employee data, attrition trends, salary distribution, department performance, and workforce insights.

---

# Project Overview

This project focuses on analyzing HR employee data using Excel tools and visualization techniques. The dashboard provides meaningful insights into employee attrition, department-wise employee distribution, salary analysis, overtime impact, and age group trends.

---

# Features

- Interactive Dashboard
- KPI Cards
- Pivot Tables
- Pivot Charts
- Slicers for Dynamic Filtering
- Employee Attrition Analysis
- Department-wise Employee Analysis
- Salary by Job Role
- Overtime vs Attrition
- Age Group Analysis

---

# KPIs Included

- Total Employees
- Attrition Count
- Attrition Rate
- Average Salary
- Active Employees

---

# Tools & Techniques Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Conditional Formatting
- Dashboard Design
- Excel Formulas

---

# Dashboard Insights

- Research & Development department has the highest number of employees
- Attrition rate is approximately 16%
- Sales Executives receive the highest average salary
- Employees aged 26–35 form the largest workforce group
- Overtime employees show higher attrition trends
